Clearing Glitches

When I first connected feel.html to my main file, it would only run when multiple options were selected in a certain manner.
I tried rearranging the order of glitches and placing in flags. However, the method that cleared the glitches entirely was to place a delay in clearGlitchesAndSwitch(). While it slows down the transition, it allows all the glitches on screen to be cleared and disabled entirely before loading the new html page. 


            function clearGlitchesAndSwitch() {
                clearGlitches();  // Clears glitches
                setTimeout(() => {
                    window.location.href = "feel.html";
                }, 500);  // 500ms delay for glitch clearing
            }
