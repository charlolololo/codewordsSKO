    function draw() {

        // Count the number of "think" and "feel" words
        let thinkCount = words.filter(word => word.text === 'think').length;
        let feelCount = words.filter(word => word.text === 'feel').length;
    
        // Change background color based on majority word
        if (thinkCount > feelCount) {
        background(white);
        } else {
        background(pink);
        }

    words.forEach(word => {

    function mousePressed() {
        // Check if a word is clicked and remove it
        for (let i = words.length - 1; i >= 0; i--) {
        let word = words[i];
        let d = dist(mouseX, mouseY, word.x, word.y);
        if (d < 50) { // Adjust the radius as needed
            words.splice(i, 1);
            break;
        }
        }
    }

SKO

We also presented our SKO ideas in class with the rest of the class. I proposed two ideas: a claw machine, and catching circles.


For my first idea, I was inspired by a Chinese Designer on Xiaohongshu. In this code, the audience would have to use their hands to grab the characters in the claw machine manually. My plan for this idea is to use the claw as the cursor, and viewers would have to grab one of the 12 different 3D characters to access each week's reflection. My second idea is based off the portfolio I made for my internship applications using Readymag. Viewers would have to chase after the individual elements on the screen, similar to my Readymag website, in order to access each page. After sharing with the class, I decided to go with the first idea as Karen Ann was enthusiastic about the idea of a claw machine. I personally also felt that it was a stronger original idea than the second one, as I have never done this before.
