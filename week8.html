Adding Sound in p5.js

Today's class focused on adding sound to our files in p5.js as someone brought it up. I thought it was interesting to learn how to add music, as I might want to use it for some of the glitches
in my final project, even though I have not gotten that far yet. Two key points to note when adding sound would be to add the p5 sound library into the folder, as well as suspend and resume audio context, a necessary function to get audio usage from browsers.

Creating a Dropdown Box

After looking at examples online, it appeared that standard practice for creating dropdown boxes was to use HTML and CSS instead of Javascript. Thus, I tried working with HTML and CSS for the first time in a while, as I had spent the past few weeks trying out my project with p5.js. It was much easier to change aesthetics of the dropdown box to match exactly what I wanted it to look like, compared to Javascript. Thus, I might keep using HTML for the main page, and only Javascript for the interactivity and glitches.
